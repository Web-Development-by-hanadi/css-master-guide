What are Display and Position?

Display & Position Properties Reference

üì¶ Display Values

  • display: block - Full width, line break
  • display: inline - Inline flow, no line break
  • display: inline-block - Inline with block properties
  • display: none - Completely removed from layout
  • display: flex - Flexbox layout
  • display: grid - Grid layout
  • display: table - Table layout
  • display: contents - No box, children visible

üìç Position Values

  • position: static - Default, in normal flow
  • position: relative - Relative to normal position
  • position: absolute - Relative to positioned ancestor
  • position: fixed - Relative to viewport
  • position: sticky - Relative until scroll threshold
  • position: inherit - Inherit from parent

üéØ Offset Properties

  • top - Distance from top edge
  • right - Distance from right edge
  • bottom - Distance from bottom edge
  • left - Distance from left edge
  • inset - Shorthand for top/right/bottom/left

üìö Stacking Properties

  • z-index - Stacking order
  • float - Float element left/right
  • clear - Clear floating elements
  • visibility - Show/hide without affecting layout
  • opacity - Element transparency

‚ö° Common Patterns

  • position: absolute; top: 0; left: 0; - Top-left corner
  • position: fixed; bottom: 0; right: 0; - Bottom-right fixed
  • position: sticky; top: 0; - Sticky header
  • display: inline-block; vertical-align: middle; - Inline alignment
  • position: relative; z-index: 1; - Create stacking context
