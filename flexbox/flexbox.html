Flexbox Properties Reference

üé™ Container Properties

  • display: flex - Enable flexbox
  • flex-direction - Main axis direction
  • justify-content - Main axis alignment
  • align-items - Cross axis alignment
  • flex-wrap - Single/multi-line
  • align-content - Multi-line alignment
  • gap - Space between items
  • row-gap - Space between rows
  • column-gap - Space between columns

üéØ Item Properties

  • order - Visual order
  • flex-grow - Ability to grow
  • flex-shrink - Ability to shrink
  • flex-basis - Initial size
  • flex - Shorthand (grow shrink basis)
  • align-self - Individual alignment

üìê Direction Values

  • row - Left to right (default)
  • row-reverse - Right to left
  • column - Top to bottom
  • column-reverse - Bottom to top

üé® Alignment Values

  • flex-start - Start of container
  • flex-end - End of container
  • center - Center of container
  • space-between - Equal space between
  • space-around - Equal space around
  • space-evenly - Equal space throughout
  • stretch - Fill container
  • baseline - Align by text baseline

üîÑ Wrap Values

  • nowrap - Single line (default)
  • wrap - Multiple lines
  • wrap-reverse - Multiple lines reversed

‚ö° Shorthand

  • flex: 1 - flex: 1 1 0
  • flex: auto - flex: 1 1 auto
  • flex: none - flex: 0 0 auto
  • flex: initial - flex: 0 1 auto
  • flex: 2 1 200px - grow shrink basis
