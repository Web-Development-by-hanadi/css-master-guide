Box Model Properties

üì¶ Box Model Core

  • width - Content width
  • height - Content height
  • margin - Outer spacing
  • padding - Inner spacing
  • border - Border styling
  • box-sizing - Box model type

üé® Margin Properties

  • margin-top - Top margin
  • margin-right - Right margin
  • margin-bottom - Bottom margin
  • margin-left - Left margin
  • margin: 10px - All sides
  • margin: 10px 20px - Vertical & horizontal
  • margin: 10px 20px 30px 40px - Top, right, bottom, left

üìê Padding Properties

  • padding-top - Top padding
  • padding-right - Right padding
  • padding-bottom - Bottom padding
  • padding-left - Left padding
  • padding: 10px - All sides
  • padding: 10px 20px - Vertical & horizontal
  • padding: 10px 20px 30px 40px - Top, right, bottom, left

üîÑ Box-Sizing

  • box-sizing: content-box - Default (adds padding/border)
  • box-sizing: border-box - Includes padding/border in width/height
  • inherit - Inherit from parent
  • initial - Default value

üéØ Border Properties

  • border-width - Border thickness
  • border-style - Border style (solid, dashed, etc.)
  • border-color - Border color
  • border-radius - Rounded corners
  • border - Shorthand property
  • border-top, border-right, etc. - Individual sides

üìè Sizing Constraints

  • min-width - Minimum width
  • max-width - Maximum width
  • min-height - Minimum height
  • max-height - Maximum height
  • overflow - Content overflow behavior
  • box-shadow - Shadow effects
