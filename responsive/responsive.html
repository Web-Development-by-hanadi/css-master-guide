Responsive Properties Reference

üì± Media Queries

  • @media (max-width: ...) - Maximum width
  • @media (min-width: ...) - Minimum width
  • @media (orientation: portrait) - Portrait mode
  • @media (orientation: landscape) - Landscape mode
  • @media (hover: hover) - Hover capable
  • @media (pointer: coarse) - Touch device
  • @media (prefers-color-scheme: dark) - Dark mode
  • @media (prefers-reduced-motion) - Reduce motion

üìê Responsive Units

  • % - Percentage of parent
  • vw - Viewport width (1vw = 1% of viewport width)
  • vh - Viewport height (1vh = 1% of viewport height)
  • vmin - Minimum of vw and vh
  • vmax - Maximum of vw and vh
  • rem - Relative to root font-size
  • em - Relative to parent font-size
  • ch - Width of "0" character

üéØ Layout Techniques

  • flex-wrap: wrap - Flexible wrapping
  • grid-template-columns: repeat(auto-fit, minmax()) - Auto grid
  • object-fit: cover - Responsive images
  • clamp(min, preferred, max) - Fluid sizing
  • min() - Minimum value
  • max() - Maximum value
  • calc() - Calculations
