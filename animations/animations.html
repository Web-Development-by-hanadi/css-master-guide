What are CSS Animations?

CSS animations allow you to animate transitions from one CSS style configuration to another, creating engaging and interactive user experiences.

üîÑ Transition Properties

  • transition-timing-function - Speed curve
  • transition-delay - Delay before start
  • transition - Shorthand property

üéØ Transform Properties

  • transform - 2D/3D transformations
  • transform-origin - Transformation origin
  • transform-style - 3D space rendering
  • perspective - 3D perspective
  • perspective-origin - Perspective origin

  • animation-name - Keyframes name

  • animation-timing-function - Speed curve

  • animation-iteration-count - Repeat count
  • animation-direction - Play direction
  • animation-fill-mode - Styles before/after
