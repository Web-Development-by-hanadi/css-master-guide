Grid Properties Reference

🔲 Container Properties

  • display: grid - Enable grid
  • grid-template-columns - Define columns
  • grid-template-rows - Define rows
  • grid-template-areas - Define areas
  • grid-template - Shorthand
  • gap - Space between items
  • row-gap - Space between rows
  • column-gap - Space between columns
  • justify-items - Align items horizontally
  • align-items - Align items vertically

🎯 Item Properties

  • grid-column-start - Start column line
  • grid-column-end - End column line
  • grid-row-start - Start row line
  • grid-row-end - End row line
  • grid-column - Column shorthand
  • grid-row - Row shorthand
  • grid-area - Area placement
  • justify-self - Self horizontal alignment
  • align-self - Self vertical alignment

📏 Unit Values

  • fr - Fractional unit
  • px - Pixels
  • % - Percentage
  • auto - Automatic sizing
  • min-content - Minimum content
  • max-content - Maximum content
  • minmax() - Size range
  • repeat() - Repeat pattern

🎨 Alignment Values

  • start - Start of container
  • end - End of container
  • center - Center of container
  • stretch - Fill container
  • space-between - Equal space between
  • space-around - Equal space around
  • space-evenly - Equal space throughout

📐 Functions

  • repeat() - Repeat track pattern
  • minmax() - Define size range
  • fit-content() - Fit to content
  • min() - Minimum value
  • max() - Maximum value
  • clamp() - Clamped value
